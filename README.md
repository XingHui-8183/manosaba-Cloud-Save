# manosaba-Cloud-Save
魔法少女的魔女审判 云存档
**本项目使用GitHub仓库实现云存档的功能**

**设计初衷是因为本人有多台电脑 希望能随时都能接着存档去玩 但是奈何此游戏没有steam云存档 本项目使用GitHub作为云存储 这样免去了配置服务器的麻烦 无论是正版玩家还有学习版玩家都能使用云存档 现将这个小工具开源出来 帮助有需要的玩家（理论上这个不止能云存档这个游戏）**

**首次使用时 请先在设置里设置Github的Token**

**存档位置会自动获取 无需手动设置 **

**准备工作**

**首先你得有一个GitHub账户（应该有的吧）然后点击这里跳转到token生成页 **[Personal Access Tokens (Classic)](https://github.com/settings/tokens) **点击生成Generate new token (classic) 即可**

**获取到token后 就可以创建一个仓库里 设置为私有就行 然后把用户名 token 以及仓库的名字复制到设置里 保存 就可以啦 **

**这个会在存档后自动上传到仓库里 当然 你也可以把存档下载下来恢复 但是恢复存档的时候请务必关闭游戏！**

**软件挂在后台就可以啦 不过上传时会稍微卡一下是正常的哦 如果连接不到GitHub请使用爬梯 **
